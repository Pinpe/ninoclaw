✨ 功能特点

智能对话

基于 OpenAI API 兼容的模型，支持自定义模型和接口地址，和 pomi 轻松聊天。

命令执行

AI 通过 command: 标签调用系统 Shell，返回执行结果。

上下文管理

自动保存对话历史，支持压缩（summary）或清空（clear），控制 token 消耗。

工作目录保持

AI 执行的命令在当前工作目录运行，支持 cd 命令（仅限绝对路径）。

记忆机制

专属 MEMORY.md 文件，写入重要信息，实现长期记忆。

技能扩展

自定义可调用的外部命令列表，AI 能通过 --help 查看用法。

📦 安装

环境要求

  • Python 3.8+
  • 兼容 OpenAI API 的模型接口（OpenAI、Azure OpenAI、vLLM 等）

安装步骤

  1. 1. 克隆仓库或下载源码
    git clone https://github.com/yourname/ninoclaw.git
    cd ninoclaw
  2. 2. 安装依赖
    pip install -r requirements.txt

    若无 requirements.txt，手动安装：

    pip install gnureadline openai rich
    gnureadline 用于改善 Linux 下 input() 体验，Windows 可安装 pyreadline3 或忽略
  3. 3. 配置技能（skill）

    手动在 shell 配置中添加别名：

    alias edit "/home/pinpe/文档/代码和项目/ninoclaw/skill/edit.py"
    alias web "python /home/pinpe/文档/代码和项目/ninoclaw/skill/web.py"
    alias get-llm "python /home/pinpe/文档/代码和项目/ninoclaw/skill/get_llm.py"
    alias vision "python /home/pinpe/文档/代码和项目/ninoclaw/skill/vision.py"

🚀 使用方法

启动程序

python main.py

启动后显示 ASCII 艺术标题和历史上下文，进入交互模式（输入提示符：>>）。

内置命令

命令 作用
summary 压缩当前上下文，保留重要信息（调用 AI 总结）
clear 清空所有上下文历史
command 手动执行Shell命令
undo/td> 删除上一条上下文
exit 退出程序
